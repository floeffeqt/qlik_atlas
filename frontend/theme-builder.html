Theme Builder (Qlik production ZIP)

Alle Theme-Properties koennen im Guided Editor mit passendem Eingabetyp bearbeitet werden (inkl. Kurzbeschreibung). Der Export bleibt produktionsfaehig: nur theme.json + <name>.qext.

QEXT Metadaten

Export bleibt unveraendert ohne Zusatzdateien.

Farb-Labor
Rundes Farbrad mit vollem Spektrum. Farbe suchen per #870414 oder rgb(135,4,20) und direkt fuer Variablen verwenden.
Helligkeit
Guided Property Editor
Variablenverwaltung: neue Variablen anlegen sowie Name/Wert bestehender Variablen bearbeiten. Umbenennen aktualisiert alle Referenzen im Theme (z. B. dataColors.primaryColor).

Control-Typ wird automatisch gewaehlt: Color Picker, Select, Zahl, Text oder Palette.
